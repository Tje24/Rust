1. ERROR[E0599]: no method named `is_some` found for unit type `()` 
-------------------------------------------------------------------
Ubicación (ejemplo): src/lib.rs:51:30
- PROBLEMA: Estás invocando `.is_some()` sobre el resultado de la función `poll_events()`, pero esta función retorna el tipo `()` (unit), que no tiene ese método.
- SOLUCIÓN:  
  a) Asegúrate de que `poll_events()` retorne un tipo `Option<T>`.  
  b) Si no debe retornar nada, elimina el uso de `.is_some()` y revisa la lógica de esa condición.
  c) Si sí debe retornar un tipo opcional, edita la función para retornar `Option<...>` correctamente.

-------------------------------------------------------------------

3. ERROR[E0308]: mismatched types (tipos incompatibles)
-------------------------------------------------------
Ubicación (ejemplo): src/core/editor_app.rs:182:46
- PROBLEMA: Estás pasando una referencia INMUTABLE a una función que espera `&mut` (mutable).
- SOLUCIÓN:  
  a) Verifica que el argumento que pasas a la función sea mutable.  
  b) Si necesitas cambiar de inmutable a mutable, usa `let mut variable...` y pásala como `&mut variable`.
  c) Asegúrate que la función también acepte la mutabilidad correctamente.

-------------------------------------------------------------------

- PROBLEMA: Tienes variables o imports que no utilizas.
